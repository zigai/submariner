# Submariner

A 2D underwater game written in Java using LibGDX and Ashley.

![screnshot](assets/screenshot.png)


### Controls:
- ```W``` - Up
- ```S``` - Down
- ```A``` - Backward
- ```D``` - Forward
- ```SPACE``` - Shoot
- ```P``` - Pause
- ```R``` - Restart
- ```F5``` -  Toogle debug grid
- ```F6``` -  Toggle hitbox indicators
